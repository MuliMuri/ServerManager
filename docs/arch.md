  - 完整名称
  - 序列号
  - 核心数

- 内存
  - 完整名称
  - 序列号
  - 容量
  - 频率
  - 物理位置 (可选)
- GPU
  - 完整名称
  - 序列号
  - 核心信息
  - 显存容量
  - 物理位置 (可选)
- 存储
  - 完整容量
  - 分区信息
  - (单硬盘)
    - 完整名称
    - 序列号
    - 容量
    - SMART
- 网络
  - 网卡数
  - 理论满载流量
  - (单网卡)
    - 完整名称
    - 序列号
    - IP
    - MAC
    - 物理位置 (可选)
